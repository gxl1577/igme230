CARY REDESIGN

library.rit.edu/cary/ is an online representation of RIT’s Cary Graphic Arts Collection, home to 40,000+ bodies of work- including artist’s books, manuscripts, print arts, illustrations, posters, and type specimens. Today, the website is around 10 years old and a redesign is considered to achieve better user experience.

Duration

10 weeks Spring 2018

Role

UI/UX, Visual, Motion

Problem

RESPONSIVENESS

Originally designed for 1920 x 1080 screen, no 4k, not mobile friendly

TARGET USERS

Lack of research tools on home page for researchers. Visual design may feel outdated for today’s visitors.

STATIC SITE

There is no motion design on the site. Lack of interactiveness makes it less interesting.

DEAD LINKS

Some links lead users to nowhere

Solution

RESPONSIVENESS

Support 4k and mobile displays

TARGET USERS

A search tool on home page for researchers. An updated design for today’s visitors.

DYNAMIC SITE

Implement motion design and interactions to make the site alive

FIX DEAD LINKS

Eliminate unused links to achieve a better user experience

Information Architecture

I created a flow map of what the redesigned website will look like.

Wireframe

A wireframe is created to visualize what the layout will be like.

Styleguide

A styleguide is created to polish the visual design. I choose RIT related colors in addition to fonts that matches Cary's asthetics.
